# Bond Market Flow & Positioning Engine

A next-level BondStats tool designed to assess institutional demand, flow pressure, auction-style stress and bond market positioning in one professional framework.

## Features

- Current-data seeded defaults
- Institutional demand proxy
- Flow pressure indicator
- Auction stress proxy
- Positioning regime engine
- Radar profile
- Sovereign yield map
- Scenario matrix

## Files

- index.html
- style.css
- script.js
